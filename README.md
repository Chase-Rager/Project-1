# Project-1
This repository contains all data, code, documentation, and reporting materials developed to analyze PurpleAir (AirPurple) sensor data across Nebraska from February 2024 through March 2025. The project evaluates particulate matter (PM2.5 and PM10) and volatile organic compounds (VOC) in relation to EPA air quality standards, meteorological conditions, and geographic factors to support public health assessment by the UNMC Water, Climate and Health Working Group.
